# Sistema nutricion
Es un sistema de nutricion que permite a los usuarios gestionar los alimentos, dietas y recomendaciones sobre alimentos balanzeados para una vida saludable
## Caracteristicas 
El proyecto lleva la implementacion de gestionar de alimentos,dietas y un sistema experto basado en reglas que realiza controles sobre los alimentos,evalua la informacion nutricional de cada alimnento y da recomendacion sobre dietas para pacientes con condiciones clinicas (ej:diabetes,hipertension,etc.)
### Clonar e instalar el proyecto

Para clonar e instalar el proyecto, seguir los siguientes pasos:

1. Clonar el repositorio con el comando `git clone https://github.com/exebrites/sistema_nutricion.git`
2. Moverse al directorio del proyecto con el comando `cd sistema_nutricion`
3. Instalar las dependencias con el comando `pip install -r requirements.txt`
6. Ejecutar el comando `python manage.py migrate` para crear la base de datos
7. Iniciar el servidor con el comando `python manage.py runserver`
